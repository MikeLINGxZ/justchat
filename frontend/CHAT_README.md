# Chat AI 聊天界面

这是一个完整的 AI 聊天界面，基于 React + TypeScript + Ant Design 开发，支持现代化的聊天体验。

## 📦 功能特性

### 🎯 核心功能
- ✅ **新建对话按钮** - 快速创建新的聊天会话
- ✅ **搜索框** - 支持搜索历史对话记录和主题
- ✅ **历史对话列表** - 显示所有历史对话，支持点击切换
- ✅ **主题分类** - 支持对话分类管理（最近对话、工作相关、学习笔记、创意写作）
- ✅ **实时聊天** - 与 AI 进行实时对话
- ✅ **Markdown 渲染** - 支持完整的 Markdown 语法，包括代码高亮
- ✅ **文件上传** - 支持图片、文档等多种文件类型上传
- ✅ **模型选择** - 支持选择不同的 AI 模型
- ✅ **响应式设计** - 完美适配桌面端和移动端

### 🎨 界面特性
- ✅ **现代化设计** - 卡片式布局、圆角、阴影等现代设计元素
- ✅ **暗黑/亮色主题** - 自动跟随系统主题切换
- ✅ **消息气泡** - 用户和 AI 消息采用不同样式区分
- ✅ **加载状态** - 优雅的加载动画和状态提示
- ✅ **消息操作** - 支持消息复制、编辑、删除等操作
- ✅ **自动滚动** - 新消息自动滚动到底部
- ✅ **快捷键支持** - Ctrl/Cmd + Enter 发送消息

### 📱 移动端适配
- ✅ **响应式布局** - 自适应不同屏幕尺寸
- ✅ **触摸优化** - 优化移动端触摸交互
- ✅ **侧边栏折叠** - 移动端智能折叠侧边栏
- ✅ **悬浮按钮** - 移动端快捷操作按钮

## 🔧 技术栈

- **框架**: React 19 + TypeScript
- **状态管理**: Zustand
- **UI组件库**: Ant Design 5.27
- **样式**: SCSS + CSS Modules
- **路由**: React Router Dom 7
- **Markdown**: react-markdown + remark-gfm
- **代码高亮**: react-syntax-highlighter
- **HTTP客户端**: Axios
- **构建工具**: Vite

## 📝 使用说明

### 基本操作
1. **开始聊天**: 点击"新建对话"按钮或直接在输入框中输入消息
2. **切换对话**: 在侧边栏点击历史对话记录
3. **搜索对话**: 使用搜索框快速找到特定对话
4. **发送消息**: 在输入框中输入文本，按 Ctrl/Cmd + Enter 发送
5. **上传文件**: 点击输入框左侧的附件按钮

### 高级功能
- **消息操作**: 鼠标悬停在消息上，点击更多按钮进行复制、编辑、删除
- **模型选择**: 点击输入框的设置按钮选择不同AI模型
- **主题切换**: 系统自动跟随或手动切换暗黑/亮色主题
- **响应式**: 自动适配桌面端和移动端界面

### 快捷键
- `Ctrl/Cmd + Enter`: 发送消息
- `Enter`: 换行（桌面端）/ 发送（移动端）
- `Shift + Enter`: 强制换行
- `Esc`: 取消编辑模式

## 🔄 状态管理

使用 Zustand 进行状态管理，主要状态包括：

- `currentChatId`: 当前激活的对话ID
- `currentMessages`: 当前对话的消息列表
- `chatList`: 所有对话列表
- `searchQuery`: 搜索查询词
- `selectedModel`: 选中的AI模型
- `isSidebarCollapsed`: 侧边栏折叠状态

## 🌐 API集成

聊天功能通过以下API接口实现：

- `POST /v1/chat/completions_result`: 创建聊天完成
- `GET /v1/chats`: 获取对话列表
- `GET /v1/chats/{chatId}/messages`: 获取对话消息
- `DELETE /v1/chats/{chatId}`: 删除对话
- `DELETE /v1/chats/{chatId}/messages/{messageId}`: 删除消息
