# 添加供应商问题修复

### 无法关闭窗口

在添加供应商窗口中，点击取消无法关闭添加供应商窗口

解决：使用 backend/service/window/window.go 中的 CloseAddProvider 方法

### 样式问题

1. step-1 选择供应商 ：移除供应商列表项右侧的圆圈标记。
2. step-2 填写供应商信息 ：
   1. 将"默认模型"字段改为"模型列表"。
   2. 模型列表占满剩余空间，支持滚动浏览。
   3. 在"模型列表"标题右侧提供刷新按钮和添加按钮。
   4. 点击添加按
   5. 钮时，在列表中新建一个输入项，输入框自动聚焦，支持输入模型名称。输入框右侧提供确认（✓）和取消（✕）按钮。
   6. **模型合并策略** ：若用户在获取模型列表前已添加自定义模型，点击刷新或保存时需合并用户已添加的模型和接口返回的模型，避免数据丢失。
3. 添加供应商页面，页面顶部的标题，左侧不需要留白
4. 设置中，添加供应商页面的子菜单宽度长点（仅改变添加供应商页面的子菜单宽度）
